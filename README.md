# 🚗 Engine Health Prediction using Machine Learning

## 📌 Project Overview

This project predicts the health of an engine (Healthy or Faulty) using machine learning techniques.
It uses sensor data such as temperature, vibration, pressure, and RPM to make predictions.

---

## 🎯 Objective

To build a simple ML model that can:

* Analyze engine sensor data
* Predict whether the engine is **Healthy ✅** or **Faulty ⚠️**

---

## 🧠 Machine Learning Model

* Algorithm Used: **Random Forest Classifier**
* Type: **Classification Problem**
* Output: Binary (0 = Healthy, 1 = Faulty)

---

## 📊 Features Used

* 🌡️ Temperature
* 📳 Vibration
* ⚙️ Pressure
* 🔄 RPM

---

## 🛠️ Technologies Used

* Python 🐍
* Scikit-learn
* Pandas
* NumPy
* Streamlit (for web app)
